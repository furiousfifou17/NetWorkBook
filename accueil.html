Présentation :



NetWorkBook est un logiciel de cahier réseau spécifiquement conçu pour les switch HP Aruba suivant :

JL075A Aruba 3810M 16SFP+ 2-slot + 2 JL084A Aruba 3810M 4-ports Stacking Module
JL253A Aruba 2930F 24G 4SFP+
JL254A Aruba 2930F 48G 4SFP+
JL255A Aruba 2930F 24G PoE+ 4SFP+
JL256A Aruba 2930F 48G PoE+ 4SFP+
JL658A Aruba 6300M 24SFP+ 4SFP56
JL724B Aruba 6200F 24G 4SFP+
JL726B Aruba 6200F 48G 4SFP+
JL725B Aruba 6200F 24G classe4 POE 4SFP+
JL727B Aruba 6200F 48G classe4 POE 4SPF+



Son premier rôle est de fournir un cahier réseau sous forme de tableau pour vos piles de switchs.
Il permet aussi de gérer l'adressage des vlans et les informations de l'établissement concernant le réseau.
Sa deuxième fonction est d'être aussi un générateur de scripts de configuration des piles de switchs. Ces scripts sont générés à partir des données du cahier réseau.
